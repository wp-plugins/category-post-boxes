== Description ==
Enables a shortcode which, when inserted in a page or post, creates a series of link boxes corresponding to the posts in a certain category.

== Installation ==
Simply copy the entire folder to your wordpress plugin folder or upload to your FTP server.
You can use the following shortcode:
[category-posts-boxes category]
where 'category' is the category slug you want.
